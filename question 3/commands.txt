Qn 3 

1. File Creation

Command = echo "this is sample data for link testing" > sample_data.txt
Explanation = to create sample_data.txt and write sample text into it

Command = cat sample_data.txt
Explanation = to display the contents of sample_data.txt

--------------------------------------------------------------------------------------------

2. Hard Link Creation

Command = ln sample_data.txt sample_hard.txt
Explanation = to create a hard link named sample_hard.txt pointing to sample_data.txt

Command = ls -l sample_data.txt sample_hard.txt
Explanation = to verify the hard link creation

-------------------------------------------------------------------------------------------

3. Symbolic Link Creation

Command = ln -s sample_data.txt sample_soft.txt
Explanation = to create a symbolic link named sample_soft.txt pointing to sample_data.txt

Command = ls -l sample_soft.txt
Explanation = to verify the symbolic link creation

----------------------------------------------------------------------------------------------------------------------------

4. Inode Verification

Command = ls -i sample_data.txt sample_hard.txt sample_soft.txt
Explanation = to display the inode numbers of the original file, hard link, and symbolic link

------------------------------------------------------------------------------------------------------------------------------

5. Inode Analysis

Explanation = sample_data.txt and sample_hard.txt share the same inode number because a hard link points to the same inode on disk. sample_soft.txt has a different inode number because a symbolic link is a separate file that only stores the path to the original file.

-------------------------------------------------------------------------------------------------------------------------------
