1. System Uptime Verification

Command = uptime -p
Explanation = to display the time elapsed since the system was last booted in a human-readable format

------------------------------------------------------------------------------------------------------------------------

2. User Process Listing

Command = ps -u $USER
Explanation = to list all processes currently running under my user account

--------------------------------------------------------------------------------------------------------------------------

3. CPU Usage Analysis

Command = ps -u $USER --sort=-%cpu
Explanation = to list all processes running under my user account sorted by CPU usage in descending order so the highest CPU-consuming process appears first

----------------------------------------------------------------------------------------------------------------------------------------

4. Background Process Execution

Command = sleep 300 &
Explanation = to start a command in the background

Command = jobs
Explanation = to verify that the background process is running

7.  Disk Space Inspection

Command = df -h ~
Explanation = This command displays the disk space usage of the filesystem where the home directory is located.
The -h option shows the size in a human-readable format.

---------------------------------------------------------------------------------------------------------------------------------------------

8. Shell Identification

Command = echo $SHELL
Explanation = This command displays the name and path of the shell that is currently being used by the user.

---------------------------------------------------------------------------------------------------------------------------------------------

9. Output Redirection

Command = uname -a > system_report.txt
Explanation = This command redirects the output of the uname -a command, which displays detailed system information, into a file named system_report.txt instead of showing it on the terminal.

Command = cat system_report.txt
Explanation = This command is used to display the contents of the system_report.txt file to confirm that the system information was successfully written to the file.

---------------------------------------------------------------------------------------------------------------------------------------------
