2. Workspace Validation

Command = pwd
Explanation =  this command tells me the directory I am currently working in

Command = ls -l
Explanation = This command shows all files and folders in the directory along with detailed information

3.  Environment Confirmation File

Command = echo "Linux user environment verified" > user_info.txt
Explanation = to create the file user_info.txt and write the given line in it

-------------------------------------------------------------------------------------------------------------------------

4. File Integrity Check

Command = wc -m user_info.txt
Explanation = to display the number of characters present in user_info.txt

--------------------------------------------------------------------------------------------------------------------------

5. Learning the Tools

Command = man mkdir
Explanation = to open the manual page of mkdir command

Command = mkdir -p test_dir/sub_dir
Explanation = the -p option allows creating parent directories if they do not exist

----------------------------------------------------------------------------------------------------------------------------

6. Home Directory Inspection

Command = ls ~ | sort
Explanation = to list the contents of my home directory sorted alphabetically

-----------------------------------------------------------------------------------------------------------------------------

7. Log Investigation

Command = grep "admin" log.txt
Explanation = to search for the word admin inside log.txt and show matching lines

8. System Information Check

Command = uname -r
Explanation = to display the linux kernel version currently running

9. Network Connectivity Test

Command = ping -c 4 www.google.com
